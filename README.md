# TCalc
Source code for an STM32 calculator. This repository contains the firmware source code for both the main and keyboard processors as well as the custom schematics.

Some notable features:
* Full math expression display engine
* Multi-function graphing
* Support for custom variable and function definitions
* Basic linear algebra capabilities
* Useful built-in functions such as linear system solver, polynomial regression, etc
* Built-in periodic table with extensive information for every element
* Snake and Tetris mini-games
* [Here's it playing Bad Apple because why not.](https://youtu.be/kNUgEzcHYEs) [[Separate repo here]](https://github.com/tylertian123/tcalc-badapple)
